# Static-Google-Clone
A static micro-interaction google.com search page.

**📖 About the Project**

This project features a clean almost indentical static google.com search page. As a beginner I made this project to learn about how a site is made how it is designed. It is a simple beginner project.

**🛠️ Tech Stack**

HTML

CSS

Git

VS Code

**🧠 What I Learned**

I learned about how HTML and CSS together work with each other. How different tags like div helps divide a website into multiple sections and making it easier to work on them individually. CSS helped in giving style to the website. I learned tools like Git and GitHub how they work. Overall this project is a stepping stone to my career in Full Stack Development. This is just a start of something new and exiciting.
